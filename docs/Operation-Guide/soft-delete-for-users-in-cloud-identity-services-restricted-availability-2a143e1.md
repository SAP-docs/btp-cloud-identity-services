# Soft Delete for Users in Cloud Identity Services \(Restricted Availability\)

Soft delete \(blocking\) for users in SAP Cloud Identity Services.

> ### Caution:  
> Soft delete for users is a restricted availability feature for SAP Cloud Identity Services released to selected customers contacted by the Cloud Identity Services team. The following options are available as restricted availability and marked as such in this documentation. No information about the soft delete can be found in the public documentation of the product.

> ### Note:  
> To have the soft delete enabled for your tenant report an incident on [SAP Support Portal Home](https://support.sap.com/en/index.html) with the *BC-IAM-IDS* component.
> 
> Please include in the incident the following information:
> 
> -   Your tenant ID from `https://tenant ID>.accounts.ondemand.com/admin` or `https://<tenant ID>.accounts.cloud.sap/admin`
> -   Your *Customer ID*
